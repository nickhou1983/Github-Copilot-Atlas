10 / CONTEXT CONSERVATION

上下文保护策略

子 Agent 分担认知负载 | 主 Agent 保留决策上下文 | 节省 70-80% 上下文窗口

核心思想

  • Atlas 只保留高层决策上下文
  • 重型操作委派给专职子 Agent
  • Explorer 消化大量代码探索结果
  • Oracle 消化外部文档和研究
  • Sisyphus 消化实现细节和测试
  • 子 Agent 返回精简摘要

精简摘要模式

  • Explorer: 返回文件列表 + 关键发现
  • Oracle: 返回技术方案 + 关键参考
  • Code-Review: 返回 APPROVED/REVISION
  • 避免传递原始代码到父 Agent
