09 / PARALLEL EXECUTION

并行执行策略

多 Agent 同时工作 | 上下文独立 | 10x 效率提升

研究阶段并行

Prometheus 同时委派:

  • Explorer → 代码库探索
  • Oracle → 外部文档研究

两者独立执行，结果合并用于规划

实现阶段并行

Atlas 可同时委派:

  • Sisyphus → 后端实现
  • Frontend-Engineer → 前端 UI

文件不重叠时可完全并行

审查阶段并行

阶段完成后 Atlas 同时启动:

  • Code-Review → 代码审查
  • Explorer → 回归检查

审查与探索并行加速反馈循环

并行安全条件

  • 各 Agent 操作的文件集不重叠
  • 不存在数据依赖关系
  • 每个 Agent 有独立上下文窗口
  • Atlas 负责合并和冲突检测
